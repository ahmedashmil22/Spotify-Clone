# Spotify Clone 🎵

A responsive front-end clone of the Spotify Web Player built using HTML5 and CSS3. This project recreates the look and feel of Spotify's desktop interface, including the sidebar, navigation bar, music player, playlists, and responsive layout.

## 🚀 Features

* Spotify-inspired user interface
* Responsive sidebar navigation
* Sticky top navigation bar
* Music player with playback controls
* Album information section
* Playlist and card layout
* Volume slider and player controls
* Responsive design using Flexbox and Media Queries

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* Font Awesome Icons
* Google Fonts (Montserrat)
